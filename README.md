# Database-Project-CSC4710
The database of Tree-Cutting Service using MySQL, JAVA, jsp. Users now can Sign up, sign in, and view users list as root user.
First, make sure that you have Tomcat 9.0 and eclipse installed and configured correctly on your PC,
Then download the zip code file, extract it in your eclipse-workspace,
Then open your eclipse, click File-Import-General-Project from Folder or Archive, and choose the extracted folder.
Right click the project folder in Porject Explorer and choose Run on server.
